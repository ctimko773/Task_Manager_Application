# Task Manager Database
*An interface that displays three different databases (Contacts, Appointments, Tasks) that give a visual animation of both bubble sort and linear search in action*

**Tech Stack**
- HTML
- CSS
- JavaScript
- Java
- SpringBoot


## Features
*When navigated to the site displays multiple databases that allow the user to see two differen algorithms in action bubble sort and linear search. Bubble sort can be seen as each entry moves to its correct possition while linear search can be seen with colored entrys yellow for searching green for found*

### Instructions
1. Clone the repo and go into folder
    - `git clone https://github.com/ctimko773/Task_Manager_Application`
    - `cd Task_Manager_Application`
2. Open project in VSCode
3. Install Extension Pack
    -"Extension Pack for Java"
4. Run TaskManagerApplication.java
5. Open the browser and go to the webpage
    - `https://localhost:8080`
